# Energy-Island

For Windows:

1) Clone the project
2) delete the current environment and create your own using py -m venv env
3) run this .\env\Scripts\activate
4) pip install -r requirements.txt
5) python manage.py runserver

For Mac:

1) Clone the project
2) delete the current environment and create your own using py -m venv env
3) run this source env/bin/activate
4) pip install -r requirements.txt
5) python manage.py runserver
